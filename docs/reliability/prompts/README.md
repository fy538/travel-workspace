# Agent Prompt Templates

These prompts are meant for Codex, Claude Code, or another coding agent working
inside the Travel Workspace. They intentionally bias toward deterministic
testing, static tracing, and mock mode so reliability work does not burn product
LLM/API tokens by default.

Use them by replacing the bracketed fields, then pasting the prompt into a new
agent session from the workspace root.
