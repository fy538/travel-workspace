## Family: atoms

**Health:** Structurally this family is in better shape than feared — the code has a real shared-atom layer (components/ui/Button, Badge, Avatar, SheetHeader with built-in 36×4 handle, Eyebrow, SectionTitle, EditorialSectionHeader) and a strong token layer (231 spreads of typography.caps*, plus a mono typography.kicker token that matches canon VKicker), which is exactly the "token, not component" mechanism the canon itself endorses for the mono micro-label.
